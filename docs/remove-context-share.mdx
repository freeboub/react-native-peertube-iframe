### Removing context menu on long-press:

Wrap the `<PeertubePlayer />` in a View that has `pointerEvents="none"` to disable app touch-events to the player.

Then react-native's [Pressable API](https://reactnative.dev/docs/pressable) or any of the touchables to intercept presses.

example:

```jsx
<Pressable
  onPress={() => {
    // handle or ignore
  }}
  onLongPress={() => {
    // handle or ignore
  }}>

  <View pointerEvents="none">
    <PeertubePlayer (...) />
  </View>

</Pressable>
```
