React Native Peertube-iframe

A wrapper of the Peertube IFrame player API build for react native. This wrapper is based on react-native-youtube-iframe You may still find some imported code or documentations inherited from this repository.

  • ✅ Works seamlessly on both ios and android platforms
  • ✅ Uses the webview player which is known to be stable
  • ✅ Supports multiple peertube player instances in a single page
  • ✅ Works on modals and overlay components
  • ✅ Expo support
  • ❌ Playlist support
  • ❌ Password support
